문제 3. 자기지도 학습 (Self-supervised Learning)

문제 분석: 2교시형 / 2문제 복합형 / 핵심 키워드: Self-supervised Learning, 데이터 효율성, 동작 방식, 활용 분야
출제 의도: 레이블링 비용 문제 해결을 위한 자기지도 학습의 부상, GPT/BERT 등 사전학습 모델의 핵심 기반 기술로서의 중요성
인간의 개입 없이 데이터 자체의 구조로부터 학습하는 자기지도 학습의 데이터 효율성과 동작 방식, 활용 분야를 중심으로 설명하겠습니다.

I. 자기지도 학습의 개요

가. 자기지도 학습의 개념

자기지도 학습(Self-supervised Learning)이란, 레이블이 없는 대규모 데이터에서 데이터 자체의 구조적 특성을 활용하여 자동으로 학습 신호(pretext task)를 생성하고, 이를 통해 범용적인 데이터 표현(representation)을 학습하는 기술

기존 지도학습은 사람이 일일이 정답(레이블)을 붙여야 하는 반면, 자기지도 학습은 데이터의 일부를 가리거나 변형하여 스스로 정답을 만들어냅니다. GPT(다음 단어 예측), BERT(빈칸 채우기) 등이 대표적입니다.

나. 등장 배경

지도학습에 필요한 대규모 레이블링 비용 문제, 비정형 데이터(텍스트, 이미지, 음성)의 폭발적 증가, 그리고 Foundation Model(기반 모델) 패러다임의 등장이 자기지도 학습을 핵심 기술로 부상시켰습니다.

ChatGPT, GPT-4 등 생성형 AI의 사전학습 단계가 바로 자기지도 학습이며, AI 산업의 패러다임 전환을 이끌고 있다.

II. 지도학습 대비 자기지도 학습의 데이터 효율성 및 특징 비교

가. 학습 방식 비교 구조

지도학습 (Supervised Learning)자기지도 학습 (Self-supervised Learning)

나. 데이터 효율성 및 특징 비교표

비교 항목지도학습 (Supervised)자기지도 학습 (Self-supervised)

학습 데이터 규모수만~수백만 건 (레이블링 병목)수억~수조 건 (인터넷 전체 데이터 활용 가능)

전이 학습제한적 (도메인 특화)강력 (사전학습 후 fine-tuning 패러다임)

대표 사례이미지 분류(ImageNet), 스팸 필터GPT(다음 토큰 예측), BERT(MLM), SimCLR(대조학습)
자기지도 학습은 데이터 효율성에서 압도적 우위를 가지며, 이것이 GPT 시리즈가 인터넷 전체 텍스트를 학습 데이터로 활용할 수 있는 핵심 이유이다. 이러한 특성을 기반으로 동작 방식을 살펴보겠다.

핵심은 Pretext Task의 설계입니다. 주요 방식은 3가지로 분류됩니다.

동작 방식원리Pretext Task 예시대표 모델
생성 기반 (Generative) 입력 데이터의 일부를 가리거나 손상시킨 후, 원래 데이터를 복원하도록 학습 빈칸 채우기(MLM), 다음 토큰 예측(CLM), 이미지 마스킹(MAE) BERT, GPT, MAE
대조 학습 (Contrastive) 같은 데이터의 다른 변환(positive pair)은 가깝게, 다른 데이터(negative pair)는 멀게 임베딩 공간에 배치 이미지 증강 후 같은 이미지끼리 유사도 최대화, 다른 이미지와는 최소화 SimCLR, MoCo, CLIP

IV. 자기지도 학습의 고려사항 및 발전방향

구분내용
고려사항대규모 컴퓨팅 자원 요구 (GPU/TPU 클러스터), 데이터 편향 전파 위험, Pretext Task와 Downstream Task 간 갭(gap) 최소화 필요, 학습 데이터의 저작권/개인정보 이슈
발전방향멀티모달 자기지도 학습 확대 (텍스트+이미지+음성 통합), 소규모 데이터에서도 효율적인 학습 방법론 연구, 도메인 특화 사전학습(의료, 법률, 과학), 에너지 효율적 학습(Green AI) 추구
기술사적 제언: 자기지도 학습은 데이터 레이블링의 한계를 근본적으로 해결하며 AI의 범용화를 가속화하고 있다. 기업은 자사 도메인 데이터를 활용한 도메인 특화 사전학습 전략을 수립하고, 동시에 데이터 거버넌스 체계를 확립하여 편향과 개인정보 이슈에 선제적으로 대응해야 한다.

II. ISO/IEC TR 29119-11의 목적 및 구성

가. 목적

ISO/IEC TR 29119-11은 ISO/IEC 29119 소프트웨어 테스팅 표준 시리즈의 Part 11로서, AI 기반 시스템의 테스팅에 대한 가이드라인을 제공하기 위한 기술 보고서(Technical Report)이다.

핵심 목적은 다음과 같습니다.

번호목적설명
1AI 테스트 가이드 제공AI 시스템의 특수성을 고려한 테스트 기법, 절차, 고려사항에 대한 실무 가이드 제공
2기존 표준과의 연계ISO/IEC 29119 시리즈(Part 1~5)의 기존 테스트 프로세스/기법을 AI에 확장 적용하는 방법 제시
3AI 품질 속성 검증정확성(Accuracy), 공정성(Fairness), 강건성(Robustness), 설명가능성(Explainability) 등 AI 고유 품질 속성의 테스트 방법 정의
4리스크 기반 접근AI 시스템의 리스크 수준에 따라 테스트 범위와 깊이를 차등 적용하는 체계 마련

나. 구성

구성항목내용
1범위 및 용어 정의AI 시스템 테스트의 범위, 적용 대상, 핵심 용어 정의 (ML, DL, 강화학습 등)
2AI 테스트 과제 식별AI 시스템의 테스트 과제 11가지: 정확성, 공정성, 강건성, 보안성, 투명성, 설명가능성, 개인정보보호, 자율성 수준, 데이터 품질, 편향, 윤리적 영향
3테스트 접근법블랙박스, 화이트박스, 경험 기반 테스트 기법의 AI 적용 방법
4AI 생명주기별 테스트데이터 수집/전처리, 모델 학습, 검증, 배포, 운영/모니터링 단계별 테스트 활동 정의
5테스트 환경 및 도구AI 테스트에 필요한 인프라, 자동화 도구, 테스트 데이터 관리 방안
ISO/IEC TR 29119-11은 AI 품질 보증의 국제 표준 프레임워크이며, 특히 블랙박스와 화이트박스 관점에서 AI 시스템에 특화된 테스트 기법을 제시한다.

III. AI 시스템의 블랙박스 테스트 기법과 화이트박스 테스트 기법

가. 블랙박스 테스트 기법 (모델 내부를 보지 않고 입출력 관계로 검증)

기법설명AI 적용 예시
동등 분할 (Equivalence Partitioning)입력 도메인을 동등한 클래스로 분할하여 대표값으로 테스트이미지 분류 시 밝은/어두운/보통 조도별 이미지 테스트
경계값 분석 (Boundary Value)입력값의 경계 부근에서 오류가 발생하기 쉬운 지점을 집중 테스트나이 예측 모델에서 유아/성인/노인 경계 나이 테스트
메타모픽 테스팅 (Metamorphic Testing)입력을 변환해도 출력 관계가 유지되는지 검증 (오라클 문제 해결의 핵심)이미지 회전/크기변경 시 분류 결과가 동일한지 검증
페어와이즈 테스팅 (Pairwise)다수 입력 변수의 조합을 최소한의 테스트 케이스로 커버자율주행 시나리오 (날씨 x 시간 x 도로 조건) 조합 테스트
적대적 테스팅 (Adversarial Testing)모델을 속이기 위한 의도적 입력(적대적 예제)으로 강건성 검증이미지에 미세 노이즈를 추가하여 오분류 유도 테스트
공정성 테스팅 (Fairness Testing)보호 속성(성별, 인종 등)에 따른 모델 출력의 편향 검증대출 승인 모델에서 성별/인종별 승인율 편차 측정

문제 6. 군집분석 (Cluster Analysis)

문제 분석: 2교시형 / 3문제 복합형 / 핵심 키워드: 군집분석, 유사도(Similarity), 비유사도(Dis-Similarity)
출제 의도: 비지도학습의 핵심인 군집분석의 기본 개념과, 유사도/비유사도 측정이 군집 품질에 미치는 영향 이해
군집분석의 개념 및 특징, 유사도(Similarity) 측정지표, 비유사도(Dis-Similarity) 측정지표를 중심으로 설명하겠습니다.

I. 군집분석의 개요

가. 군집분석의 개념

군집분석(Cluster Analysis)이란, 사전에 정의된 레이블 없이 데이터 간의 유사도(Similarity)를 기반으로 유사한 데이터를 동일 그룹(군집)으로, 상이한 데이터를 다른 그룹으로 분류하는 비지도학습(Unsupervised Learning) 기법

핵심 원리는 "군집 내 유사도 최대화(Within-cluster similarity), 군집 간 비유사도 최대화(Between-cluster dissimilarity)"입니다.

비유: 도서관에서 책을 분류하는 것과 같습니다. 정해진 분류 기준(레이블)이 없어도, 비슷한 주제의 책끼리 모아두면 자연스럽게 "소설", "과학", "역사" 같은 그룹이 형성됩니다. 같은 그룹의 책은 유사하고, 다른 그룹의 책은 상이합니다.

특징설명

II. 군집의 유사도(Similarity) 측정지표

가. 유사도의 개념

유사도(Similarity)란, 두 데이터 객체 간의 닮은 정도를 수치로 표현한 것으로, 값이 클수록 두 데이터가 유사함을 의미한다. 일반적으로 0~1 사이의 값으로 정규화된다.

나. 유사도 측정지표

측정지표수식/원리특징적합 데이터
코사인 유사도 (Cosine Similarity) 두 벡터 간 각도의 코사인 값. cos(theta) = (A . B) / (||A|| x ||B||). 값 범위: -1~1 벡터의 크기(magnitude)보다 방향(direction)을 비교. 고차원 희소 데이터에 적합 텍스트(TF-IDF), 문서 유사도, 추천시스템
자카드 유사도 (Jaccard Similarity) 두 집합의 교집합 / 합집합. J(A,B) = |A 교 B| / |A 합 B|. 값 범위: 0~1 집합 기반 비교, 이진(binary) 데이터에 적합 문서 키워드 집합, 상품 구매 여부, 유전자 서열
피어슨 상관계수 (Pearson Correlation) 두 변수의 선형 상관관계. r = cov(X,Y) / (std(X) x std(Y)). 값 범위: -1~1 선형 관계의 강도와 방향을 측정. 평균 중심화(centering) 효과 연속형 수치 데이터, 평점 데이터
SMC (Simple Matching Coefficient) 전체 속성 중 일치하는 속성의 비율. SMC = (일치 수) / (전체 속성 수) 이진 데이터에서 0-0 일치도 유사로 간주 이진 속성 데이터 (설문 응답 등)
다이스 계수 (Dice Coefficient) 두 집합의 겹침을 강조. D = 2|A 교 B| / (|A| + |B|). 값 범위: 0~1 자카드보다 교집합에 더 큰 가중치. 불균형 데이터에 유리 이미지 세그멘테이션, 의료 영상 비교
유사도가 "닮은 정도"를 측정한다면, 비유사도는 "다른 정도"를 측정한다. 두 관점은 상호 보완적이며, 군집분석에서는 비유사도(거리)를 더 직접적으로 활용하는 경우가 많다.

III. 군집의 비유사도(Dis-Similarity) 측정지표

가. 비유사도의 개념

비유사도(Dis-Similarity)란, 두 데이터 객체 간의 다른 정도를 수치로 표현한 것으로, 주로 거리(Distance)로 표현되며, 값이 클수록 두 데이터가 상이함을 의미한다. 비유사도 = 1 - 유사도 관계가 성립하는 경우가 많다.

나. 비유사도(거리) 측정지표

측정지표수식/원리특징적합 데이터
유클리디안 거리 (Euclidean Distance) 두 점 사이의 직선 거리. d = sqrt(sum((xi - yi)^2)) 가장 직관적인 거리. 스케일에 민감하므로 정규화 필요 연속형 수치, K-Means의 기본 거리
맨해튼 거리 (Manhattan Distance) 각 좌표 차이의 절대값 합. d = sum(|xi - yi|) 격자 도로처럼 축 방향으로만 이동하는 거리. 이상치에 덜 민감 고차원 데이터, 이상치 존재 시
민코프스키 거리 (Minkowski Distance) 유클리디안과 맨해튼의 일반화. d = (sum(|xi - yi|^p))^(1/p) p=1이면 맨해튼, p=2이면 유클리디안. p값으로 거리 특성 조절 범용 (p값 조절로 다양한 상황 대응)
체비셰프 거리 (Chebyshev Distance) 각 차원의 차이 중 최대값. d = max(|xi - yi|) 어느 한 차원이라도 크게 다르면 비유사. 민코프스키의 p=무한대 최악의 경우(worst case) 기반 판단
마할라노비스 거리 (Mahalanobis Distance) 공분산 행렬을 고려한 거리. d = sqrt((x-y)' S^(-1) (x-y)) 변수 간 상관관계와 분산을 고려. 스케일 불변. 타원형 분포에 적합 상관관계 높은 다변량 데이터, 이상치 탐지
해밍 거리 (Hamming Distance) 같은 위치에서 값이 다른 비트/속성의 수 범주형/이진 데이터에 적합. 문자열 비교에 활용 이진 데이터, DNA 서열, 오류 검출
자카드 거리 (Jaccard Distance) 1 - 자카드 유사도. Jd = 1 - |A 교 B| / |A 합 B| 유사도의 보수(complement)로 비유사도 표현 집합 데이터, 문서 비교

다. 유사도 vs 비유사도 비교

비교 항목유사도 (Similarity)비유사도 (Dis-Similarity)
의미두 객체가 얼마나 닮았는가두 객체가 얼마나 다른가
값의 방향높을수록 유사높을수록 상이 (=거리가 멂)
관계비유사도 = 1 - 유사도 (정규화된 경우)
대표 지표코사인 유사도, 자카드 유사도, 피어슨 상관유클리디안, 맨해튼, 마할라노비스, 해밍
주 활용추천시스템, 정보 검색, 문서 유사도군집분석(K-Means 등), 이상치 탐지, KNN

IV. 고려사항 및 발전방향

구분내용
고려사항데이터 유형(연속형/범주형/이진)에 적합한 측정지표 선택, 변수 스케일 차이에 따른 정규화/표준화 선행 필수, 고차원 데이터에서의 차원의 저주 문제 대응(차원 축소 PCA/t-SNE 등), 이상치의 영향을 고려한 로버스트(robust) 거리 선택
발전방향딥러닝 기반 표현 학습(Representation Learning)으로 의미적 유사도 측정 고도화, 도메인 특화 거리 메트릭 학습(Metric Learning), 대규모 데이터에서의 근사 최근접 이웃(ANN) 기법 활용, 그래프 기반 유사도 측정 연구 확대
기술사적 제언: 군집분석의 성패는 적절한 유사도/비유사도 측정지표의 선택에 달려 있다. 데이터의 특성(유형, 분포, 차원)을 면밀히 분석한 후 최적의 측정지표를 선택하고, 필요 시 도메인 지식을 반영한 맞춤형 거리 함수를 설계하여 군집 품질을 극대화해야 한다.

4문제 핵심 요약

  • 자기지도 학습: 레이블 없이 데이터 자체에서 학습 신호를 생성하는 기술. 생성 기반(GPT), 대조 학습(SimCLR), 예측 기반(RotNet) 3가지 방식. Foundation Model의 핵심.
  • AI 테스트: ISO/IEC TR 29119-11은 AI 고유 품질(정확성, 공정성, 강건성, 설명가능성) 검증 가이드. 블랙박스(메타모픽, 적대적 테스팅)와 화이트박스(뉴런 커버리지, XAI)를 병행.
  • 멀티모달 품질검증: 일반 품질특성(정확성, 완전성, 일관성) + 모달리티별 품질특성 + 모달리티 간 정합성. 유형별 자동/수동/교차 검증 수행.
  • 군집분석: 유사도(코사인, 자카드, 피어슨)는 "닮은 정도", 비유사도(유클리디안, 맨해튼, 마할라노비스)는 "다른 정도". 데이터 유형에 맞는 지표 선택이 핵심.
