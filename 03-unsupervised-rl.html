1. 비지도학습 개요

비지도학습의 정의

비지도학습은 레이블이 없는 데이터로부터 내재된 구조와 패턴을 발견하는 학습 방식입니다. 지도학습과 달리 정답이 제공되지 않으므로, 알고리즘이 데이터 자체의 특성을 이용해 유의미한 정보를 추출해야 합니다.

비유: 낯선 도시에 도착한 관광객이 안내도 없이 도시를 탐험하면서 "아, 이 지역은 상업지구이고, 저 지역은 주거지역이네"라고 스스로 판단하는 것과 같습니다.

비지도학습 vs 지도학습

구분 지도학습 비지도학습
데이터 입력 + 정답 라벨 입력만 제공, 정답 없음
목표 정답 예측 능력 데이터 구조 파악
예시 이메일 스팸 분류 고객 세분화
평가 정확도로 측정 가능 해석적 평가 필요

비지도학습의 주요 작업

1. 군집분석(Clustering): 비슷한 특성을 가진 데이터를 그룹화합니다. 예: 고객 세분화, 문서 분류
2. 차원축소(Dimensionality Reduction): 고차원 데이터를 저차원으로 변환하여 계산 효율을 높입니다. 예: PCA, t-SNE
3. 연관규칙(Association Rules): 데이터 항목 간의 연관성을 발견합니다. 예: "맥주를 사면 기저귀도 산다"
4. 이상탐지(Anomaly Detection): 정상 패턴과 다른 이상값을 찾아냅니다.

적응식 신경망

비지도학습에서 신경망은 입력 데이터의 구조에 적응하여 스스로 학습합니다. 경쟁학습(competitive learning)을 통해 유사한 입력에 반응하는 뉴런을 강화하고, 다른 입력에는 약하게 반응하도록 조정됩니다.

2. 거리와 유사도 측정

군집분석의 핵심: 거리와 유사도

군집분석은 데이터 포인트 간의 거리 또는 유사도를 기반으로 그룹을 형성합니다. 거리는 데이터가 얼마나 다른지를, 유사도는 얼마나 비슷한지를 나타냅니다.

군집분석의 조건:
- 집단 내 동일성(Within-cluster similarity): 같은 군집 내 데이터는 비슷해야 함
- 집단 간 이질성(Between-cluster dissimilarity): 다른 군집 간 데이터는 달라야 함

비유사도(거리) 측정 방법 - 5가지

거리 방식 수식 설명
유클리드 거리
(Euclidean)		 √(Σ(x_i - y_i)²) 직선거리. 가장 일반적이며 피타고라스 정리 기반. 이상치에 민감함.
맨해튼 거리
(Manhattan)		 Σ|x_i - y_i| 도시 블록 거리. 격자를 따라 이동하는 거리. 고차원에서 효율적.
민코프스키 거리
(Minkowski)		 (Σ|x_i - y_i|^p)^(1/p) 유클리드와 맨해튼의 일반화. p값으로 거리 타입 조정 가능.
표준화 거리
(Standardized)		 √(Σ((x_i - y_i)/σ_i)²) 각 차원의 표준편차로 정규화. 단위 차이 제거.
마할라노비스 거리
(Mahalanobis)		 √((x-y)^T Σ^(-1)(x-y)) 데이터 분포를 고려. 공분산 행렬 사용. 상관관계 반영.
기억법: "유맨민표마" = 유클리드, 맨해튼, 민코프스키, 표준화, 마할라노비스

유사도 측정 방법 - 상관계수 3가지

상관계수 범위 특징
피어슨 상관계수 -1 ~ 1 선형 관계 측정. 연속 변수용. 가장 널리 사용됨.
스피어만 상관계수 -1 ~ 1 순위 기반. 비선형 관계도 포착. 이상치에 강함.
켄달 타우 상관계수 -1 ~ 1 순위 일치도. 작은 표본에 적합. 계산 비용 높음.

유사도 측정 방법 - 기타 유사도 4가지

유사도 방식 범위 사용분야
코사인 유사도 -1 ~ 1 문서 유사도, 추천 시스템. 벡터 간 각도 측정.
자카드 계수 0 ~ 1 집합 유사도. 공통 항목의 비율. 쇼핑 장바구니 비교.
레벤슈타인 거리
(편집거리)		 0 이상 문자열 유사도. 오타 교정. 한 문자열을 다른 것으로 변환하는 최소 연산 횟수.
해밍 거리 0 이상 같은 길이 문자열 비교. DNA 서열. 다른 위치의 개수.
기억법: "코자레해" = 코사인, 자카드, 레벤슈타인, 해밍
거리와 유사도의 관계: 거리가 크면 유사도는 작고, 거리가 작으면 유사도는 큽니다. 실제 응용에서는 데이터의 특성과 목표에 맞는 측정 방식을 선택하는 것이 중요합니다.

3. 군집분석 알고리즘

군집 알고리즘의 분류

분류 유형 특징 대표 알고리즘
계층적 응집형
(Agglomerative)		 하향식. 개별 포인트에서 시작해 점진적으로 병합 단일연결, 완전연결, 평균연결
계층적 분할형
(Divisive)		 상향식. 전체 집합에서 시작해 점진적으로 분할 이분 분할(Bisecting)
비계층적 프로토타입 기반
(Prototype-based)		 군집 중심점을 기반으로 분할. 빠르고 효율적. K-Means, K-Medoids
비계층적 밀도 기반
(Density-based)		 데이터 밀도를 기반으로 군집 형성. 임의 모양 군집 감지. DBSCAN, OPTICS
비계층적 분포 기반
(Distribution-based)		 확률 분포를 가정. 소프트 할당. GMM
신경망 기반 경쟁학습 신경망이 데이터 구조 자동 학습 SOM(코호넨맵)

K-Means 알고리즘

정의: 각 군집의 중심(중균값)을 기반으로 데이터를 K개의 군집으로 분할하는 알고리즘입니다.
1
초기화
K개의 초기 중심점을 무작위로 선택
2
할당
각 포인트를 가장 가까운 중심에 할당
3
중심 업데이트
각 군집의 평균값으로 중심 갱신
4
수렴
중심이 변하지 않을 때까지 2-3 반복
비용함수(손실함수):
J = Σ Σ ||x - c_k||² (모든 포인트와 중심 간 거리의 제곱의 합)
K-Means는 이 비용함수를 최소화하려고 반복 수행합니다.
장점: 빠르고 구현이 간단, 대규모 데이터 처리 효율적
단점: 초기값에 민감, 구형 군집에만 효과적, 이상치에 약함, K값 미리 결정 필요

K-Medoids 알고리즘

정의: K-Means와 유사하지만, 중심점(centroid)이 아닌 실제 데이터 포인트 중 하나(medoid)를 중심으로 사용합니다.
1
Medoid 선택
실제 데이터 중 K개를 무작위로 선택
2
할당
각 포인트를 가장 가까운 medoid에 할당
3
Medoid 교환
군집 내에서 비용을 최소화하는 새 medoid 선택
4
비용 개선
비용이 더 이상 개선되지 않을 때까지 반복
5
수렴 검증
변경이 없으면 완료
6
결과 반환
최종 군집 할당 반환
특성 K-Means K-Medoids
중심 Centroid (가상의 중심) Medoid (실제 데이터 포인트)
이상치 강건성 약함 강함
계산 복잡도 O(nkd) O(n²kd)
해석성 낮음 높음 (실제 데이터)
K-Medoids의 장점: 이상치 영향 최소화, 해석이 용이, 모든 거리 메트릭 사용 가능
단점: K-Means보다 느림, 대규모 데이터에 부담

DBSCAN 알고리즘

정의: 밀도 기반 공간 군집화 알고리즘. 데이터가 밀집된 영역끼리 묶고, 고립된 포인트는 노이즈로 분류합니다.

DBSCAN의 핵심 개념

핵심점(Core)
epsilon 반경 내에 MinPts개 이상의 포인트를 포함하는 점
경계점(Border)
핵심점의 epsilon 반경 내에 있지만, 자신은 핵심점이 아닌 점
노이즈(Noise)
어느 군집에도 속하지 않는 고립된 점
주요 파라미터:
- Epsilon (ε): 이웃으로 간주하는 반경
- MinPts: 핵심점이 되기 위해 필요한 최소 이웃 개수
DBSCAN의 장점: 임의 모양 군집 감지, 이상치 자동 감지, K값 미리 결정 불필요
단점: Epsilon과 MinPts 선택 어려움, 밀도 차이나는 군집에 약함, 고차원 데이터에서 성능 저하

OPTICS 알고리즘

정의: DBSCAN을 개선한 알고리즘. 다양한 밀도를 가진 군집을 자동으로 감지합니다. 각 포인트의 "도달가능거리"를 계산하여 밀도 계층을 파악합니다.
OPTICS의 특징: Epsilon 값에 덜 민감, 계층적 군집화 결과 제공, 다양한 밀도의 군집 처리 가능, 계산 비용 DBSCAN보다 높음

4. 고급 군집 기법

GMM (Gaussian Mixture Model)

정의: 각 군집이 가우시안(정규) 분포를 따른다고 가정하는 확률 모델입니다. K-Means의 확률적 버전으로 볼 수 있습니다.

GMM의 핵심 파라미터 3개

평균(μ)
각 가우시안 분포의 중심
공분산(Σ)
분포의 퍼짐 정도 및 방향
혼합비율(π)
각 가우시안 분포의 가중치

EM 알고리즘 (기댓값-최대화)

정의: GMM의 파라미터를 학습하기 위한 반복적 알고리즘입니다. 숨겨진 변수(각 포인트의 군집 확률)를 다루는 데 효과적입니다.
E
E-Step (기댓값)
각 포인트가 각 가우시안에 속할 확률 계산
M
M-Step (최대화)
그 확률을 기반으로 파라미터 업데이트
EM 알고리즘의 특징:
- 소프트 할당: 각 포인트가 모든 군집에 확률적으로 속함
- Log-Likelihood: 최대화하려는 목적함수
- 수렴성: 반복할수록 Log-likelihood가 증가
- K-Means보다 느리지만 더 정교한 결과

계층적 군집화 (Hierarchical Clustering)

응집형 계층적 군집화 4단계:
1. 각 포인트를 개별 군집으로 시작
2. 가장 가까운 두 군집 찾기
3. 두 군집을 병합
4. 2-3단계를 하나의 군집 될 때까지 반복

연결법 5가지

연결법 정의 특징
단일연결
(Single)		 두 군집 간 최단 거리 체인 효과 발생 가능, 긴 가지 모양 생성
완전연결
(Complete)		 두 군집 간 최장 거리 소형 응집 군집 생성, 이상치에 민감
평균연결
(Average)		 두 군집 간 평균 거리 균형잡힌 결과, 가장 널리 사용
중심연결
(Centroid)		 두 중심점 간 거리 직관적, 계산 효율적
워드연결
(Ward)		 분산 증가 최소화 가장 좋은 성능, 비용 높음
덴드로그램(Dendrogram): 계층적 군집화 과정을 트리 구조로 시각화한 것. 아래에서 위로 올라갈수록 더 큰 군집이 됩니다.
계층적 군집화의 장점: 구조 시각화, K값 미리 결정 불필요, 해석 용이
단점: 계산 비용 O(n²~n³), 번복 불가능 (한번 병합하면 분리 불가)

SOM (자기조직화지도, 코호넨맵)

정의: 신경망 기반 비지도 학습. 입력 데이터를 2D 격자 형태로 변환하여 시각화하면서 동시에 군집화합니다.

SOM의 6단계

1
초기화
격자 뉴런의 가중치를 무작위로 초기화
2
입력 선택
학습 데이터에서 입력 벡터 선택
3
BMU 찾기
최적 일치 단위(Best Matching Unit) 계산
4
이웃 결정
BMU 근처의 뉴런들을 이웃으로 결정
5
가중치 업데이트
BMU와 이웃의 가중치를 입력에 근접하게 조정
6
반복
학습률과 이웃반경 감소시키며 2-5단계 반복
BMU (최적 일치 단위): 입력 벡터와 가장 유사한 뉴런. 이 뉴런과 주변 뉴런이 함께 학습되어 "경쟁학습"이 일어납니다.
비유: SOM은 신경망이 무용수처럼 데이터의 리듬과 패턴에 맞춰 스스로 정렬되는 것과 같습니다. 데이터와 유사한 뉴런이 활성화되고, 주변 뉴런도 함께 움직입니다.

5. 차원축소와 차원의 저주

차원의 저주 (Curse of Dimensionality)

정의: 데이터의 차원(특징 개수)이 증가하면서 발생하는 문제들:
차원의 저주의 현상:
1. 데이터 희소성: 고차원에서 데이터는 매우 흩어져 있음
2. 거리 의미 상실: 모든 포인트가 비슷한 거리를 가짐
3. 과적합 증가: 특징이 많을수록 과적합 위험 증가
4. 계산 복잡도 증가: 연산량 기하급수적 증가
5. 모델 성능 저하: "더 많은 데이터" 필요

차원축소의 목적과 기법

기법 방식 특징 사용 시점
PCA
(주성분분석)		 선형 변환 분산 최대 방향 찾기, 해석 용이 연속 변수, 선형 구조
LDA
(선형 판별분석)		 선형 변환 클래스 구분에 최적, 지도학습용 분류 문제, 레이블 있을 때
t-SNE 비선형 변환 2D/3D 시각화 우수 데이터 시각화, 탐색
UMAP 비선형 변환 t-SNE보다 빠름, 멀리까지 거리 보존 대규모 데이터, 시각화
오토인코더 신경망 복잡한 비선형 관계 학습 비정형 데이터, 심층 구조

PCA (Principal Component Analysis)

정의: 데이터의 분산이 최대인 방향을 찾아 데이터를 그 방향으로 투영합니다. 원래 100개 특징을 10개 주성분으로 축약 가능합니다.
비유: 3D 물체를 2D 평면에 사진 찍을 때, 물체의 특징이 가장 잘 드러나는 각도를 선택하는 것과 같습니다.

t-SNE (t-Distributed Stochastic Neighbor Embedding)

정의: 고차원 데이터의 국소적 구조를 보존하면서 2D 또는 3D로 축소합니다. 시각화에 매우 효과적입니다.
t-SNE의 특징:
- 비선형 차원축소
- 시각적 군집화 우수
- 계산 비용 높음
- 파라미터 튜닝 필요 (perplexity, learning_rate)

신경망 기반 차원축소

오토인코더(Autoencoder): 입력을 받아 작은 크기의 병목층으로 압축한 후 다시 복원하는 신경망. 병목층의 표현을 새로운 특징으로 사용합니다.
VAE (변분 오토인코더): 확률적 오토인코더. 새로운 데이터도 생성 가능합니다.

군집 평가지표

실루엣 계수 (Silhouette Coefficient)

정의: 각 포인트가 자신의 군집에 잘 속하는지, 다른 군집과는 잘 분리되는지 측정합니다.
범위: -1 ~ 1 (1에 가까울수록 좋은 군집화)

팔꿈치 기법 (Elbow Method)

정의: K값을 변화시키며 비용함수값을 그래프로 그린 후, 급격히 꺾이는 지점을 최적 K로 선택합니다.

K-NN vs K-Means vs K-Medoids vs GMM 비교

알고리즘 유형 비용함수 할당 이상치 속도
K-NN 분류 거리 기반 경상적 민감 느림
K-Means 군집화 유클리드 거리 경상적 (하드) 약함 빠름
K-Medoids 군집화 거리 합계 경상적 (하드) 강함 느림
GMM 군집화 확률 분포 확률적 (소프트) 중간 중간

6. 이상탐지(Anomaly Detection)

이상탐지의 정의와 필요성: 이상탐지는 정상적인 패턴에서 벗어나는 비정상 데이터를 식별하는 비지도학습 기법입니다. 대부분의 데이터가 정상이고 비정상 샘플이 극소수인 불균형 데이터 문제를 다룹니다. 사기 탐지, 시스템 모니터링, 품질 검사 등에서 필수적입니다.

이상탐지의 유형

유형 영문명 특징 예시
점 이상 Point Anomaly 개별 데이터가 정상 범위를 벗어남 신용카드 한 건의 이상거래
맥락 이상 Contextual Anomaly 특정 상황/문맥에서만 이상 겨울철 과도한 전력 사용
집단 이상 Collective Anomaly 데이터의 부분집합이 집단적으로 이상 특정 시간대 네트워크 DDoS 공격

주요 알고리즘

1. 통계적 방법 (Statistical Methods)

Z-score: 데이터가 평균으로부터 표준편차의 몇 배 떨어져 있는지 계산. |Z| > 3인 점을 이상으로 판정(정규분포 가정).

IQR(Interquartile Range): Q1, Q3 사분위수를 이용하여 경계값 설정. Q1-1.5×IQR 또는 Q3+1.5×IQR 범위 밖은 이상으로 판정. 이상치에 강건함.

2. 거리 기반 (Distance-based)

LOF(Local Outlier Factor): 이웃과의 거리 비율로 이상도 계산. 전역 이상뿐만 아니라 국소 이상도 탐지. k-NN과 유사하나 밀도를 고려한 개선 버전.

3. 밀도 기반 (Density-based)

DBSCAN 활용: 밀도 기반 군집 알고리즘으로, 어떤 군집에도 속하지 않는 점을 이상으로 판정. 임계값(eps, min_samples) 설정이 중요. 다양한 밀도의 군집 탐지 가능.

4. 앙상블 기반

Isolation Forest: 특성을 랜덤하게 선택하여 데이터를 분할하는 결정 트리 집합. 이상치는 정상보다 적은 분할로 고립되므로, 짧은 경로 길이로 탐지. 고차원 데이터에 효율적.

5. 딥러닝 기반

오토인코더 기반 이상탐지: 정상 데이터만으로 학습하여 인코더-디코더 구조. 재구성 오류(Reconstruction Error)가 크면 이상으로 판정. 복잡한 비선형 패턴 학습 가능.

활용 사례

주요 응용 분야:
  • 금융 사기 탐지: 신용카드 거래 내역에서 불정상 거래 패턴 식별
  • 네트워크 침입 탐지: 비정상적인 트래픽, 포트 스캔, DDoS 공격 탐지
  • 제조 품질 검사: 센서 데이터에서 불량품 자동 판별
  • 의료 진단: 이상 심박수, 뇌파 패턴 감지
  • 시스템 모니터링: 서버 성능 저하, 디스크 용량 급증 탐지

평가 지표 (불균형 데이터 고려)

이상탐지는 정상 샘플이 대부분이므로 Accuracy는 의미 없음. 대신:

  • Precision(정밀도): 탐지된 이상 중 실제 이상의 비율. 거짓 양성 최소화.
  • Recall(재현율): 실제 이상 중 탐지된 비율. 놓친 이상 최소화. 보안은 높은 Recall 필요.
  • F1 점수: Precision과 Recall의 조화평균. 불균형 데이터 평가에 적합.
  • ROC-AUC: 임계값 변화에 따른 성능 곡선 평가.

쉽게 말하면

비유: 병원 응급실의 환자 분류와 같습니다. 대부분의 환자는 가벼운 질환이지만(정상), 드물게 생명을 위협하는 중증 환자가 들어옵니다(이상). 의료진은 일반적인 패턴에서 벗어나는 증상을 빠르게 찾아내야 합니다. 이상탐지도 정상 데이터의 "숨은 구조"를 학습한 후, 그에 맞지 않는 데이터를 찾는 것입니다.
핵심 요약:
  • 이상탐지는 극소수의 비정상 샘플을 찾는 비지도학습 문제
  • 통계, 거리, 밀도, 앙상블, 딥러닝 등 다양한 알고리즘 활용
  • 금융, 보안, 제조, 의료 등 실무 응용 매우 광범위
  • 불균형 데이터이므로 Precision, Recall, F1 지표 중요

7. 강화학습 기초

강화학습의 정의

강화학습은 에이전트가 환경과 상호작용하면서 시행착오를 통해 최적의 정책(행동 방식)을 학습하는 방식입니다. 좋은 행동에는 보상을, 나쁜 행동에는 벌칙을 주어 점진적으로 성능을 개선합니다.

비유: 개 훈련과 같습니다. 좋은 행동을 하면 간식으로 보상하고, 나쁜 행동을 하면 혼내서 학습시킵니다. 강화학습도 동일한 원리입니다.

강화학습의 9개 구성요소

에이전트(Agent)
학습하고 행동하는 지능 개체
환경(Environment)
에이전트가 상호작용하는 외부 세계
상태(State)
현재의 상황을 나타내는 정보
행동(Action)
에이전트가 선택할 수 있는 선택지
보상(Reward)
행동의 결과에 대한 즉시 피드백
정책(Policy)
상태에서 행동을 결정하는 전략
가치함수(Value)
상태나 행동의 장기적 가치 평가
수익(Return)
미래 보상의 할인된 합계
동작원리

→

강화학습의 분류

Model-Free
(비모델 기반)		 환경의 규칙을 모름. 경험으로 직접 학습 Q-Learning, DQN, Policy Gradient
Model-Based
(모델 기반)		 먼저 환경의 규칙을 학습한 후 계획 수립 MCTS, Dyna-Q
Actor-Critic 두 신경망: 정책(배우)과 가치(평론가) A3C, PPO, TRPO

8. Q-Learning과 DQN

Q-Learning (오프-폴리시 가치 기반)

정의: 각 상태-행동 쌍(state-action pair)의 가치 Q(s,a)를 학습하는 알고리즘입니다. 현재의 최적 행동이 뭔지 배웁니다.

Q-함수와 벨만 방정식

Q-함수: Q(s,a) = 상태 s에서 행동 a를 했을 때의 예상 보상
벨만 최적 방정식:
Q(s,a) = R(s,a) + γ × max Q(s',a')
- R(s,a): 즉시 보상
- γ (감마): 할인율 (0~1, 미래 보상을 얼마나 중시할지)

Q-Learning 학습과정

1
Q-table 초기화
모든 Q값을 0으로 설정
2
상태 관찰
에이전트가 현재 상태 확인
3
탐험-활용
ε-탐욕(epsilon-greedy): 무작위 행동 또는 최고 Q값 행동
4
Q값 업데이트
Q(s,a) ← Q(s,a) + α[r + γmax Q(s',a') - Q(s,a)]
Q-Learning의 특징:
- Off-policy: 현재 정책이 아닌 최고 행동으로 학습
- 수렴성: 충분히 반복하면 최적 정책에 수렴
- Q-table 필요: 상태 공간이 작을 때만 실용적
Q-Learning의 장점: 간단하고 직관적, 수학적으로 증명된 수렴성
단점: Q-table이 메모리를 많이 차지 (상태가 많으면 불가능), 연속 상태 불가능

DQN (Deep Q-Network)

정의: Q-Learning에서 Q-table 대신 신경망(특히 CNN)을 사용합니다. 게임 같은 복잡한 상황에서도 작동합니다.

Q-table vs 신경망

특성 Q-table 신경망
저장 테이블에 저장 가중치로 학습
상태 공간 작은 이산 공간 매우 큰 연속 공간
메모리 상태 수에 비례 신경망 크기에 비례
속도 빠름 느림
응용 간단한 문제 이미지 입력 (게임 등)

DQN의 핵심 기술

1. 리플레이 메모리(Experience Replay): 과거의 경험들을 메모리에 저장했다가, 무작위로 샘플링해서 학습합니다. 시간 상관관계를 제거해 학습 안정성을 높입니다.
2. 타깃 네트워크(Target Network): 두 개의 신경망을 사용합니다. 하나는 행동 선택에, 하나는 Q값 계산에 사용하여 불안정성을 줄입니다.
DQN의 혁신: Atari 게임 같은 이미지 입력을 받는 복잡한 환경에서 인간 수준의 성능을 달성했습니다. 강화학습의 실용화를 가능하게 했습니다.

SARSA (On-Policy 가치 기반)

정의: State-Action-Reward-State-Action. Q-Learning과 유사하지만, 현재 정책으로 선택한 다음 행동을 기반으로 학습합니다.
특성 Q-Learning SARSA
정책 Off-policy On-policy
다음 행동 최고 Q값 (탐욕적) 현재 정책으로 선택
특징 더 공격적, 위험 감수 더 보수적, 안전
사용 배우 환경에서 학습 실제 환경에서 즉시 적용

9. 정책 기반 강화학습

정책 기반 강화학습의 원리

가치함수를 학습하는 대신, 정책 자체를 직접 학습합니다. "빨간 신호에서는 멈춰"라는 규칙을 직접 배우는 방식입니다.

비유: 가치 기반은 "이 위치는 좋지 않다"고 평가하는 것이고, 정책 기반은 "이 상황에서는 이렇게 행동하라"고 직접 지시하는 것입니다.

Policy Gradient 방법

정의: 정책의 파라미터를 경사상승법으로 개선합니다. 보상을 최대화하는 방향으로 정책을 조정합니다.
대표 알고리즘:
- REINFORCE: 기본 정책 경사
- Actor-Critic: 정책 + 가치함수 결합
- PPO (Proximal Policy Optimization): 신뢰 구간 설정
- TRPO (Trust Region Policy Optimization): 정책 변화 제약

PPO (Proximal Policy Optimization)

정의: 정책 업데이트를 너무 크게 하지 않도록 신뢰 구간을 설정하는 방법입니다. 안정성과 효율성을 모두 갖춥니다.

PPO의 학습과정

1
경험 수집
정책으로 상호작용하여 경험 모음
2
장점 계산
각 행동의 상대적 가치 평가
3
정책 개선
신뢰 구간 내에서 정책 업데이트
4
반복
1-3단계를 여러 에폭 동안 반복
PPO의 특징: OpenAI의 현재 주력 알고리즘, ChatGPT 학습에 사용, 안정적이고 샘플 효율 높음

RLHF (Human Feedback 기반 강화학습)

정의: 인간의 피드백을 기반으로 AI 모델을 강화학습으로 미세조정합니다. 생성형 AI를 인간의 선호도에 맞추는 데 사용됩니다.
기본 모델
→
텍스트 생성
→
인간 평가
→
보상 함수 학습
→
RLHF
RLHF의 절차:
1. 인간이 생성된 텍스트 쌍을 비교
2. 선호도 데이터로 보상 함수 학습
3. 보상 함수를 기반으로 정책 최적화 (PPO)
4. 결과적으로 인간 선호도에 맞는 모델 완성
비유: 영화배우를 감독이 지도하는 것과 같습니다. 배우(모델)가 연기(텍스트 생성)를 하고, 감독(인간)이 평가하면, 배우가 점점 감독 마음에 드는 연기를 하게 됩니다.

10. 고급 강화학습 기법

모델 기반 강화학습 (Model-Based)

환경의 규칙을 먼저 학습한 후 계획을 세웁니다. "빨간 신호에서 계속 가면 사고 난다"는 인과관계를 미리 파악합니다.

방식:
1. 학습된 모델: 데이터로부터 환경 규칙 학습
2. 주어진 모델: 처음부터 환경 규칙을 앎

MCTS (Monte Carlo Tree Search)

정의: 가능한 미래를 모의 시뮬레이션해서 최선의 행동을 찾습니다. AlphaGo에서 사용된 기법입니다.
1
Selection
탐색할 유망한 노드 선택
2
Expansion
트리에 새 노드 추가
3
Simulation
끝까지 무작위로 진행하여 결과 관찰
4
Backpropagation
결과를 역으로 전파하여 노드 가치 업데이트
MCTS의 특징: 모든 가능성을 계산하지 않고 유망한 경로만 탐색, UCB (Upper Confidence Bound)로 탐험-활용 균형

AlphaGo

정의: MCTS와 심층신경망을 결합한 시스템. 무한한 가능성의 바둑 게임에서 인간 최고수를 이겼습니다.
AlphaGo의 구성:
- 정책 신경망: 유망한 수를 추천
- 가치 신경망: 위치 평가
- MCTS: 이 신경망들을 활용해 탐색 효율화

액터-크리틱 (Actor-Critic)

정의: 두 신경망이 협력합니다. 액터(정책)는 행동을 결정하고, 크리틱(가치)은 그 행동을 평가합니다.
역할 역할 기능
액터(Actor) 배우 행동 선택 (정책), 무대에서 연기
크리틱(Critic) 영화평론가 행동 평가 (가치함수), 평론 제시
비유: 무도 경연에서 참가자(액터)가 춤을 추고, 심사위원(크리틱)이 점수를 주는 것과 같습니다. 심사평이 좋아질 때까지 참가자는 춤을 개선합니다.
액터-크리틱의 장점: 정책과 가치를 동시에 학습, 더 효율적이고 안정적, 많은 현대 RL 알고리즘의 기반

연합학습 (Federated Learning)

중앙 서버에서 학습하지 않고, 각 기기(병원, 은행 등)에서 분산으로 학습한 후 결과만 모아 통합합니다.

1
초기화
서버가 초기 모델 배포
2
분산 학습
각 클라이언트가 로컬 데이터로 학습
3
파라미터 전송
클라이언트가 학습된 파라미터만 서버로 전송
4
집계
서버가 파라미터를 평균 또는 병합
5
반복
1-4단계를 반복하여 전역 모델 개선
연합학습 알고리즘:
- FedSGD: 기울기(gradient) 평균
- FedAVG: 가중치(parameter) 평균, 가장 널리 사용
- FedProx: 기기 간 차이 고려
- FedOpt: 최적화 기법 적용
- FedReID: 개인정보 보호 강화
비유: 여러 병원의 환자 데이터로 AI를 훈련할 때, 환자 정보를 중앙으로 모으지 않고 각 병원에서 학습한 결과만 모아서 합치는 방식입니다.
연합학습의 이점: 개인정보 보호, 데이터 프라이버시 유지, 규제 준수 용이
과제: 통신 오버헤드, 이질성 데이터 처리, 동기화 어려움

온디바이스 AI

정의: 클라우드가 아닌 스마트폰, 스마트워치 등 단말기에서 직접 AI 모델을 실행합니다.

하드웨어 기술

AI 가속기:
- TPU (Tensor Processing Unit): Google의 AI 전용 칩
- GPU (Graphics Processing Unit): Nvidia의 병렬 처리 칩
- NPU (Neural Processing Unit): 신경망 전용 칩

소프트웨어 경량화 기술

1. 양자화(Quantization): 가중치를 낮은 정밀도(32-bit → 8-bit)로 저장
2. 가지치기(Pruning): 중요하지 않은 뉴런 제거
3. 지식 증류(Knowledge Distillation): 큰 모델의 지식을 작은 모델로 전달
4. 레이어 병합: 여러 레이어를 하나로 통합
온디바이스 AI의 이점: 낮은 지연시간, 개인정보 보호, 오프라인 작동, 배터리 절약

강화학습의 어려움

핵심 문제: 보상 설정의 어려움
잘못된 보상을 설정하면 AI가 예상과 다르게 행동할 수 있습니다.
예: 게임에서 좋은 점수를 원했는데, AI가 버그를 이용해 부정행위로 점수를 올리는 방법을 발견할 수 있습니다.
강화학습 평가지표:
- 누적 보상(Cumulative Reward): 에피소드의 총 보상
- 평균 보상(Average Reward): 스텝당 평균 보상
- 성공률(Success Rate): 목표 달성 빈도
- 수렴성: 최적 정책에 도달하는지 여부

배치 학습 vs 온라인 학습

특성 배치 학습 온라인 학습
데이터 모든 데이터를 한 번에 스트림으로 계속 들어옴
학습 시점 학습 후 배포 실시간으로 지속 학습
적응성 낮음 높음
메모리 많음 적음
사용 이미지 분류 등 추천 시스템, 로봇

핵심 요점 정리

  1. 비지도학습은 정답 없이 데이터의 내재된 구조를 발견합니다.
  2. 거리 측정(유클리드, 맨해튼, 마할라노비스)과 유사도(피어슨, 코사인, 자카드)는 군집화의 기초입니다.
  3. K-Means는 빠르지만 이상치에 약하고, K-Medoids는 강건하지만 느립니다.
  4. DBSCAN은 임의 모양 군집을 찾고, GMM은 확률적 접근으로 소프트 할당합니다.
  5. 계층적 군집화는 덴드로그램으로 시각화되며, 반복 불가능합니다.
  6. SOM은 신경망 기반으로 2D 격자에 데이터 구조를 매핑합니다.
  7. 차원의 저주는 고차원에서 발생하는 여러 문제이며, PCA, t-SNE, 오토인코더로 해결합니다.
  8. 강화학습은 보상과 벌칙으로 최적의 정책을 배웁니다.
  9. Q-Learning은 가치를 학습하고(Off-policy), SARSA는 정책을 따릅니다(On-policy).
  10. DQN은 신경망으로 Q-Learning을 확장하여 게임 등 복잡한 환경에서 우수합니다.
  11. PPO와 정책 경사는 정책을 직접 최적화하여 더 안정적입니다.
  12. 액터-크리틱은 정책과 가치를 동시에 학습하는 강력한 방식입니다.
  13. MCTS와 AlphaGo는 모델 기반 강화학습의 성공 사례입니다.
  14. RLHF는 인간의 선호도에 따라 생성형 AI를 미세조정합니다.
  15. 연합학습은 개인정보를 보호하면서 분산 환경에서 학습합니다.
  16. 온디바이스 AI는 TPU, GPU, NPU 같은 하드웨어와 양자화, 가지치기 같은 소프트웨어 기술을 활용합니다.
